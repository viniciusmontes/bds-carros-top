# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/viniciusmontes/bds-carros-top

# entrar na pasta do projeto front end web
cd frontweb

# instalar dependências
yarn install

# executar o projeto
yarn start
```
